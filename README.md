## 📚 Key Learnings & Insights

### Common Mistakes I Made (And Fixed!)
1. **Case Sensitivity** - Linux is case-sensitive (`Desktop` ≠ `desktop`)
2. **Spaces in Commands** - `mkdir~training` vs `mkdir training`
3. **Permission Errors** - Scripts need execute permission (`chmod +x`)
4. **grep -n vs grep -c** - Line numbers vs count
5. **Path Operations** - Can create files in subdirs without `cd`

### Biggest Breakthroughs
1. Understanding permission structure (rwx + numeric)
2. Mastering pipes for command chaining
3. Command substitution `$()` for dynamic scripting
4. Error redirection `2>/dev/null` for clean output
5. Creating reusable bash scripts
