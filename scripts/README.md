# 🛠️ Scripts Collection

Custom tools created during learning journey.

## log_analyzer.sh
**Purpose:** Analyzes log files for ERROR, WARNING, and INFO entries.

**Usage:**
```bash
./log_analyzer.sh filename.log
```

**Features:**
- Counts each log level
- Displays ERROR details
- Works with any standard log format

## search_tool.sh
**Purpose:** Searches for patterns with line numbers.

**Usage:**
```bash
./search_tool.sh PATTERN
```

**Features:**
- Pattern matching with grep
- Match counting
- Line number display

## victory_celebration.sh
**Purpose:** Displays Phase 1 completion statistics.

**Usage:**
```bash
./victory_celebration.sh
```

**Features:**
- Skills acquired summary
- Statistics display
- Achievement tracking
